# Weather App

Built this weather app as a front-end project to strengthen my JavaScript skills and gain hands-on experience working with APIs. Users can search for cities and instantly view real-time weather information. The app features a clean, responsive interface designed to make weather forecasts quick and easy to access.

## Features

- Real-time weather data
- Smart city search suggestions
- Current temperature and weather conditions
- Humidity and wind speed information
- Responsive design for desktop and mobile
- Error handling for invalid searches

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- OpenWeather API
